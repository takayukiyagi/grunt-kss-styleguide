## ファイル構成

```
.
├── grunt
│     ├── README.md
│     ├── package.json
│     ├── Gruntfile.coffee
│     ├── node_modules
│     └── src
│       　　└─ assemble
│       　　└─ sass
│       　　└─ styleguide
│       　    　└─ index.html
│       　　    └─ public
└── htdocs
      ├── index.html
      ├── css
      └── styleguide
        　　└─ index.html
        　　└─ section-buttons.html
        　　└─ public
```

## インストール

```
cd ~/path/to/project
git clone git@github.com:takayukiyagi/grunt-kss-styleguide.git
npm install
grunt listen
```
